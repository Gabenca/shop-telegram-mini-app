## Отличия от плана

### Незначительные отличия (адаптация к новым версиям)

1. **Angular 18 соглашения:**
   - План: `app.component.ts`, `app.component.html`, класс `AppComponent`
   - Реализация: `app.ts`, `app.html`, класс `App`
   - **Причина:** Angular 18 использует новые соглашения для standalone components

2. **Gradle версия:**
   - План: Gradle 8.10.2
   - Реализация: Gradle 8.14.2
   - **Причина:** Обновление для поддержки JDK 25

3. **TelegramService типы:**
   - Коммит `48a7729`: Исправлены типы для совместимости с @twa-dev/sdk v7+
   - **Причина:** Обновление SDK
